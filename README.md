# node-red

## mac install node

```
brew install node
```

validate node

```
node --version

v14.3.0
```

validate npm
```
npm --version

6.14.4
```

## mac install node-red

```
npm install -g --unsafe-perm node-red
```

start node-red
```
node-red
```

access the node-red in browser,  http://127.0.0.1:1880/
